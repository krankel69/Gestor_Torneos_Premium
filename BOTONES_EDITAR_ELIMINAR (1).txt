PROBLEMAS ENCONTRADOS
════════════════════════════════════════════════════════════════════════════════

1. ❌ PUNTOS EN 0
   Causa: _recalcular_grupos() filtra por fase='Fase de grupos' pero partidos 
          probablemente tienen fase diferente o NULL
   Fix: Cambiar filtro a fase IS NULL O fase LIKE '%grupo%' 

2. ❌ JUGADORES NO SE VEN
   Causa: Probablemente no se llama a _refresh_jugadores_copa() después de cargar
   Fix: Agregar _refresh_jugadores() en _refresh_all() para Mundial

3. ❌ NO HAY BOTÓN EDITAR RESULTADO
   Necesario: Poder cambiar goles después de grabar

4. ❌ NO HAY BOTÓN ELIMINAR RESULTADO
   Necesario: Poder limpiar un resultado y volver a goles=-1

5. ❌ NO HAY BOTÓN ELIMINAR GOLEADOR
   Necesario: Poder eliminar un goleador específico sin borrar todo

CÓDIGO NECESARIO
════════════════════════════════════════════════════════════════════════════════

1. EDITAR RESULTADO
   - Método: _on_editar_resultado()
   - Acción: Abre diálogo con entry para nuevos goles
   - Validación: Verifica que goleadores no superen goles
   - Guarda: UPDATE partidos SET goles_local=?, goles_visitante=?

2. ELIMINAR RESULTADO
   - Método: _on_eliminar_resultado()
   - Acción: Pide confirmación
   - Limpia: UPDATE partidos SET goles_local=-1, goles_visitante=-1
   - Limpia goleadores: DELETE FROM goleadores_partido
   - Recalcula: _recalcular_grupos()

3. ELIMINAR GOLEADOR
   - Método: _on_eliminar_goleador_partido()
   - Tabla: Treeview de goleadores con columna borrar
   - Acción: Click en icono 🗑 de una fila
   - Elimina: DELETE FROM goleadores_partido WHERE id=?
   - Actualiza: Refresca tabla y suma de goles
